
# webScrapper

Built a simple web scrapping program using go with colly Framework 
## Features

- Web Scrapping 
- Developed with go 
- Saving json locally


## Run Locally

- Clone the project

```bash
  git clone https://github.com/irahuldogra/webScrapper.git
```

- Go to the project directory

```bash
  cd webScrapper
```

- Get dependencies

```bash
  go get
```

- Run Program

```bash
  go run .
```
